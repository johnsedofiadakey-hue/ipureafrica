# iPure Website Project

This is a complete, localized air purification website for the iPure brand in Ghana.

## Features
- **Modern Design**: Responsive, elegant UI tailored for African environments.
- **Localized**: Prices in Ghana Cedis (₵).
- **Admin Panel**: Full control over product listings and site settings.
- **Real-time Sync**: Changes in the admin panel update the website instantly via `localStorage`.

## Directory Structure
- `index.html`: Main website.
- `admin.html`: Control panel.
- `logo.png`: Brand logo.

## Admin Panel Access
- **URL**: Open `admin.html` in any browser.
- **Default Username**: `admin`
- **Default Password**: `ipure2025`

## Usage Tips
- **Price Toggle**: Go to **Settings** in the Admin Panel to show or hide product prices on the main site.
- **Product Editing**: Add, remove, or edit products in the **Products** tab. Changes are saved locally to your browser and will persist across sessions.
